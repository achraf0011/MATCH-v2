# مدارك التعليمية — منصة تعليمية (أولى إعدادي)

موقع تعليمي عربي (RTL) مع دروس فيديو، PDF، تمارين، وامتحانات تجريبية.

## التشغيل

- **الوصول المباشر:** افتح `index.html` في المتصفح (يعمل محليًا، الخطوط قد لا تُحمّل بدون إنترنت).
- **خادم محلي (موصى به):** شغّل خادمًا ثابتًا من مجلد المشروع، مثلاً:
  - `npx serve .`
  - أو `python -m http.server 8080`
  - ثم افتح `http://localhost:...` لتفعيل Service Worker والعمل دون اتصال بعد التحميل الأول.

## هيكل المشروع (الإنتاج)

```
chat-app/
├── index.html          ← نقطة الدخول
├── sw.js               ← Service Worker للتخزين المؤقت والعمل دون اتصال
├── css/
│   └── main.css        ← كل الأنماط
├── js/
│   ├── config.js       ← إعدادات ومفاتيح التخزين
│   ├── utils.js        ← أدوات (esc, now)
│   ├── storage.js      ← IndexedDB لملفات PDF + localStorage
│   ├── auth.js         ← تسجيل دخول المسؤول (مع الاستمرار)
│   ├── modals.js       ← النوافذ المنبثقة والتنبيهات
│   ├── app.js         ← حالة التطبيق والتوجيه
│   ├── navbar.js      ← شريط التنقل (قابل لإعادة الاستخدام)
│   ├── chat.js        ← الدردشة مع استمرار localStorage
│   ├── pages.js       ← عرض الصفحات ومعالجات النماذج
│   └── bootstrap.js   ← ربط الأحداث والبدء
├── assets/             ← (قديم، غير مستخدم في index.html)
└── docs/
```

## المميزات

- **شريط تنقل واحد** يُحمّل مرة واحدة ويعمل في كل الأقسام.
- **دردشة عامة** مع حفظ المحادثات في `localStorage`.
- **ملفات PDF** مخزّنة في IndexedDB؛ التحميل/الفتح يعمل بعد التحديث.
- **دخول المسؤول** يُحفظ في `localStorage` ويبقى عبر الصفحات والتحديث.
- **يعمل دون اتصال** (بعد التحميل الأول عند التشغيل عبر خادم محلي) بفضل Service Worker.
